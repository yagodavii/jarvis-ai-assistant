# Morgan — PM

**Modelo:** Sonnet 4.6
**Papel:** Product Manager do [[AIOX-CORE — Agentes]]
**Tier:** Produto e planejamento

---

## Responsabilidades

- Escrever PRDs (Product Requirements Documents)
- Criar e executar Epics
- Priorizacao com MoSCoW e RICE
- Spec Pipeline — orquestracao
- Requirements gathering

## Comandos Principais (EXCLUSIVOS)

- `*execute-epic` — Executar um epic
- `*create-epic` — Criar novo epic
- `EPIC-{ID}-EXECUTION.yaml` — Gerenciar execucao de epics
- Spec writing (spec pipeline)

## Priorizacao

| Framework | Quando Usar |
|-----------|-------------|
| **MoSCoW** | Must/Should/Could/Won't — features de produto |
| **RICE** | Reach x Impact x Confidence / Effort — backlog geral |

## Delegacao

| Delega Para | O Que |
|-------------|-------|
| [[Agentes/River — SM]] | Criar stories a partir do PRD |
| [[Agentes/Aria — Architect]] | Assessment de complexidade |
| [[Agentes/Atlas — Analyst]] | Pesquisa de mercado/tecnica |

## Quando Usar

- Definir um novo produto/feature
- Criar PRD ou epic
- Priorizar backlog
- Spec Pipeline — fases 1 e 4
- Brownfield Discovery — fase 10
