# Uma — UX

**Modelo:** Sonnet 4.6
**Papel:** UX Designer do [[AIOX-CORE — Agentes]]
**Tier:** Design e experiencia do usuario

---

## Responsabilidades

- Design de interfaces e experiencia do usuario
- Sistema de design tokens
- Atomic Design (atoms -> molecules -> organisms -> templates -> pages)
- Acessibilidade e usabilidade

## 5 Fases de Trabalho

| Fase | O Que |
|------|-------|
| 1. **Research** | Pesquisa de usuario, benchmarks |
| 2. **Audit** | Auditoria de UI/UX existente |
| 3. **Tokens** | Design tokens (cores, tipografia, espacamento) |
| 4. **Build** | Construcao de componentes |
| 5. **Quality** | Validacao de qualidade visual e funcional |

## Atomic Design

```
Atoms -> Molecules -> Organisms -> Templates -> Pages
(botao)  (search bar) (header)    (layout)    (home)
```

## Quando Usar

- Nova interface precisa ser desenhada
- Auditoria de UX existente
- Criar sistema de design tokens
- Brownfield Discovery — fases 3 e 6
- Qualquer trabalho visual/frontend design
